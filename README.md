# 🎬 MovieMatrix Pro – Intelligent Film Matching System

## AI-Powered Movie Recommendation System

MovieMatrix Pro is a Machine Learning-based web application that predicts whether a user is likely to enjoy a movie based on personal preferences and movie attributes. The application is built using **Python**, **Streamlit**, and **Scikit-learn**, providing a simple, interactive, and user-friendly interface.

This project demonstrates the practical use of Machine Learning in building an intelligent movie recommendation system by analyzing user behavior, movie genres, ratings, and viewing patterns.

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/MovieMatrixPro.git
```

Go to the project directory:

```bash
cd MovieMatrixPro
```

Install required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run web.py
```
